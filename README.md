
# Reto Módulo 05 - Pacientes COVID

## Descripción
En este reto se crea una base de datos de pacientes COVID y se desarrollan varias consultas SQL para obtener estadísticas y análisis sobre los pacientes registrados.

## Requisitos previos
- Gestor de base de datos compatible (PostgreSQL o SQL Server recomendado).
- Herramienta para ejecutar scripts SQL (DBeaver, PgAdmin, SSMS, etc.).
- Archivos: `reto.sql` y `Reto_mod_05_-_pacientes_covid.sql`.

## Instrucciones de uso
1. Abre tu gestor de base de datos y selecciona una base de datos vacía o crea una nueva.
2. Ejecuta el archivo `reto.sql` para crear la tabla e insertar los datos de ejemplo.
3. Ejecuta las consultas del archivo `Reto_mod_05_-_pacientes_covid.sql` para obtener los distintos resultados y estadísticas.

## Consultas incluidas
- Listar pacientes por fase.
- Calcular promedios de edad y días enfermo.
- Buscar pacientes por rango de edad y fase.
- Filtrar pacientes por rut y hospital.
